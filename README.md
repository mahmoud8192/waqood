# Waqood

## Company Websit (Frontend Only)

A multi-page company website featuring Home, News & Events, About Us, Careers, Brands, and Contact Us. 
This project is a frontend implementation using HTML, CSS, and JavaScript with a focus on responsive design and user experience.

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6+)
- Responsive Design (Mobile-first)
